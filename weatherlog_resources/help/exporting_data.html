Exporting Data

WeatherLog can export the data entered, for if you want to have a backup of the data or share it with someone else. To export data, either click the save button on the toolbar, select Export from the File menu, or press Control-S. Then enter the filename to use for the exported file.

However, this is only really useful if the only application you want to use this data with is WeatherLog. That's why there are two more options, for exporting data to HTML or CSV. To export to CSV or HTML, click on the respective button in the file chooser window. Data will be formatted properly in the choosen format before it is saved.

If you want to easily share the data with others, you can also export the data to Pastebin, as either raw data or formatted as HTML or CSV.

Note that the export feature only works with the data stored in the current profile, not all of them.
